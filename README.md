# Vanilla TypeScript Store

A functional web application for browsing products and managing a shopping card, built with Vanilla TypeScript.

## Features 
- Product Catalog: Dynamic product rendering with pagination.
- Shopping Cart: Add, remove, and track selected items using localstorage.
- Search & Filter: Find products by ID and category.
- Responsive design: Clean and modern UI built with CSS flexbox.

  ## Tech Stack

  - Language: TypeScript
  - Styling: CSS3
  - Data: JSON-based mock API
